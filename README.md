# Alioth Marketing Solutions

**South African Marketing Solutions — Real People. Real Results. Real Systems.**

Marketing landing page for Alioth Marketing Solutions, built as a single-file HTML page using the Alioth brand system.

## Brand System

| Token | Value |
|---|---|
| Alioth Black | `#0E0D0B` |
| Alioth Cream | `#F1E9DA` |
| Deep Forest | `#1B3B2F` |
| Primary font | Cormorant Garamond 600 |
| Body font | Archivo 400 |
| Label font | Space Mono 700 |

## Sections

- **Hero** — Display headline with grid background and dual CTAs
- **Marquee** — Animated services ticker
- **Problem** — Three core industry pain points
- **Services** — Learn It · Automate It · Build It Properly
- **Who We Serve** — Four audience segments
- **Approach** — Four-step process (accordion)
- **About** — Team positioning + brand story
- **Accent** — "Alioth is the star you steer by" (Deep Forest)
- **Contact** — Strategy call booking form
- **Footer**

## Deployment

This site is deployed via **GitHub Pages** from the `main` branch root.

Live URL: [https://rdoubell.github.io/Alioth-Marketing-](https://rdoubell.github.io/Alioth-Marketing-)

No build step. Single `index.html` — fonts loaded via Google Fonts CDN. Open directly in any browser.
